## Purpose

This document tracks weekly action plans and progress for all active roles across the ops team. It is the single source of truth for what each pod is working on, week by week.

**Updated by:** Hrishikesh + Senior OMs | **Review cadence:** Weekly (every Monday)

**Related docs:** [[02-team-structure|Team Structure]] · [[10-pipeline-management|Pipeline Management]] · [[13-senior-om-playbook|Senior OM Playbook]]

---

## How to Use This Tracker

1. **Each week gets its own section** below, headed by the week's start date
2. **Roles are grouped by account**, with the assigned Senior OM noted
3. **Action items** are listed as checkboxes — mark complete as you go
4. **Status** is updated at end of week: ✅ Done · 🟡 In Progress · ❌ Blocked
5. **New weeks are added at the top** so the latest plan is always first
